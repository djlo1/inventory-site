FAQ
Questions fréquentes
Tout ce que vous devez savoir pour démarrer avec Inventory.

Ouvrez l'application et cliquez « Créer un compte ». Renseignez le nom et le numéro de votre assemblée (format XX0000), puis créez votre profil avec un nom d'utilisateur et un code PIN à 4 chiffres.

Réglages → Membres → « + Ajouter ». Renseignez prénom, nom, nom d'utilisateur, code PIN et rôle. Les permissions se gèrent dans l'onglet Accès.

Mouvements → Arrivage. Entrez le numéro de packing list, ajoutez les articles manuellement ou via import Excel. Un modèle téléchargeable est disponible.

Stock → Revues ou Cahiers → « + Nouveau numéro ». Les anciens numéros s'archivent automatiquement quand le stock atteint 0, ou manuellement.

Inventaire → sélectionnez le mois → « 📄 PDF ». Les articles archivés sont masqués automatiquement.

Commandes → « Générer ». Le système analyse seuils, sorties, demandes spéciales et inter-assemblées pour proposer une liste ajustable, exportable en PDF.

Écran connexion → « PIN oublié ? ». Entrez nom d'utilisateur, nouveau PIN et numéro assemblée pour vérification.
